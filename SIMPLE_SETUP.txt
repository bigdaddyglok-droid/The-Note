STEP 2: Extract It
================================================================================

On Mac/Linux:
-------------
1. Open Terminal
2. Go to where you downloaded it:
   cd ~/Downloads

3. Extract:
   tar -xzf the-note-complete.tar.gz

On Windows:
-----------
1. Right-click the-note-complete.tar.gz
2. Click "Extract All..."
3. Choose a folder

================================================================================
STEP 3: Install Python (If you don't have it)
================================================================================

Mac:
----
brew install python3

Windows:
--------
Download from: https://www.python.org/downloads/
- Check "Add Python to PATH"

Linux:
------
sudo apt install python3 python3-pip

================================================================================
STEP 4: Install Dependencies
================================================================================

Open Terminal/Command Prompt in the extracted folder:

cd backend
pip install -r requirements.txt

(This downloads PyTorch, NumPy, FastAPI - takes ~5 minutes)

================================================================================
STEP 5: Start The Note
================================================================================

In the backend folder, run:

python -m uvicorn src.thenote_backend.main:app --host 0.0.0.0 --port 8000

You'll see:
INFO:     Uvicorn running on http://0.0.0.0:8000

‚úÖ The Note is RUNNING!

================================================================================
STEP 6: Access It
================================================================================

On YOUR laptop:
---------------
Open browser: http://localhost:8000/docs

On YOUR FRIEND's laptop (same WiFi):
------------------------------------
1. Find your IP address:

   Mac/Linux:  ifconfig | grep "inet "
   Windows:    ipconfig

2. Share with friend: http://YOUR_IP:8000/docs
   Example: http://192.168.1.100:8000/docs
